## Execution
- Clone project
  - https://github.com/cheritaousley/MEANProject_AHOY-.git
- Move into express app
  - cd MEANProject_AHOY-
- run 'nodemon server.js' 
- move into angular app
  - cd angular-app
- run 'ng build --watch'
- run mongodb
